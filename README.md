# Glass - The webshop

An ecommerce site, built with Next Js/Typescript, Stripe, and a headless CMS. The client is publishing and selling photobook through the website

## Technologies

#### Web-development:

- Next Js
- Typescript
- Condentful CMS
- Stripe Api

#### Design:

- Figma
- Adobe Photoshop
- Adobe Illustrator
